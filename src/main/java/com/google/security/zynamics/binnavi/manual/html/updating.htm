Updating from BinNavi 3.X

The database support for MySQL was dropped starting with BinNavi 4.0 due to licensing issues with Oracle. The MySQL database backend has been replaced with a PostgreSQL backend.

Updating BinNavi 4.X

For all versions within 4.X BinNavi supports automatic updating of existing databases.

Updating BinNavi 4.X to 5.X

An update script is provided for updates from the 4.0.6/7 Database layout to the 5.0.0 database layout. Be advised that there have been substantial changes to the database layout, please backup your data before proceeding with the update.
